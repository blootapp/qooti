# Use the Chrome extension

1. Install the **qooti / bloot** extension from the [Chrome Web Store](https://chrome.google.com/webstore) (link on the product page).
2. Reload the browser or confirm the extension icon appears in the toolbar.
3. Use the extension action on a page to save content; details sync with collections in the desktop app where applicable.
4. If something breaks, update Chrome and the extension.
