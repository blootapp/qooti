# Install qooti on macOS

1. Open the [macOS download page](/download/thanks/macos) and download the `.dmg`.
2. Open the disk image and drag **qooti** into your **Applications** folder.
3. On first launch, macOS may block the app — allow it in **System Settings → Privacy & Security**.
4. For updates, follow in-app prompts or check [GitHub Releases](https://github.com/blootapp/qooti-releases/releases).
