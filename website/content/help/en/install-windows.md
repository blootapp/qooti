# Install qooti on Windows

1. Open the [Windows download page](/download/thanks/windows) and download the `.exe` installer.
2. Run the installer and follow the steps.
3. If **SmartScreen** warns you, you can proceed when you trust the publisher.
4. For updates, check [GitHub Releases](https://github.com/blootapp/qooti-releases/releases).

Questions? [Telegram](https://t.me/blootsupport).
