# qootini Windows’ga o‘rnatish

1. [Yuklab olish sahifasiga](/download/thanks/windows) o‘ting va `.exe` o‘rnatuvchini yuklab oling.
2. O‘rnatuvchini ishga tushiring va ko‘rsatmalarga amal qiling.
3. Agar **SmartScreen** ogohlantirishi chiqsa, nashrlovchini bilganingizda **Baribir ishga tushirish** tugmasidan foydalanishingiz mumkin.
4. Yangilanishlar uchun [GitHub relizlar](https://github.com/blootapp/qooti-releases/releases) sahifasini tekshiring.

Savollar uchun [Telegram](https://t.me/blootsupport).
