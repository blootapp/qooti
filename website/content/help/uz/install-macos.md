# qootini macOS’ga o‘rnatish

1. [Yuklab olish sahifasiga](/download/thanks/macos) o‘ting va `.dmg` faylini yuklab oling.
2. Disk tasvirini oching va **qooti** ilovasini **Programmalar** papkasiga sudrab o‘tkazing.
3. Birinchi marta ishga tushirganda macOS xavfsizlik ogohlantirishi chiqishi mumkin — **Tizim sozlamalari → Maxfiylik va xavfsizlik** bo‘limida ilovaga ruxsat bering.
4. Yangilanishlar mavjud bo‘lsa, ilova sizga xabar beradi yoki [GitHub relizlar](https://github.com/blootapp/qooti-releases/releases) sahifasidan tekshiring.

Savollar uchun [Telegram](https://t.me/blootsupport).
