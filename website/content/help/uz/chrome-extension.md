# Chrome kengaytmasidan foydalanish

1. [Chrome Web Store](https://chrome.google.com/webstore) dan **qooti / bloot** kengaytmasini o‘rnating (havola mahsulot sahifasida).
2. Brauzerni qayta yuklang yoki kengaytma ikonkasini tekshiring.
3. Kontentni saqlash uchun sahifada kengaytma tugmasidan foydalaning; tafsilotlar ilovadagi to‘plamlar bilan sinxronlashadi.
4. Muammo bo‘lsa, brauzer va kengaytma versiyasini yangilang.

Yordam: [Telegram](https://t.me/blootsupport).
