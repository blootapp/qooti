Migrate from Telegram
Import your Telegram chat history into Qooti
Export your data from Telegram Desktop in three steps, then select the folder to start importing your media into Qooti.
How it works
1
Export your chat history from Telegram Desktop
Open Telegram Desktop → go to your chat or channel → click the ⋯ menu → select Export Chat History. Enable media files (photos, videos, GIFs) before confirming.
2
Leave the exported folder exactly as it is
Don't rename, move, or modify any files. Qooti needs the folder to contain result.json alongside your media files to process the import correctly.
3
Select the folder to begin importing
Click Select folder below, choose the exported folder, and Qooti will automatically read and import your messages and media.
Supported: images, videos, and GIFs from Telegram exports.
