## Flaticon icons (drop-in)

This folder is wired into the Settings vertical tabs:

- `interface.svg`
- `interaction.svg`
- `downloads.svg`
- `discovery.svg`
- `tags.svg`
- `data.svg`

### How to use

1. Download your chosen SVG icons from `flaticon.com`.
2. Rename them to match the filenames above.
3. Replace the placeholder files in this folder.

### Notes

- The UI assumes **monochrome** (or mostly monochrome) SVGs for best results.
- If your SVGs are black, the UI applies a CSS filter to render them as light icons on the dark background.
- If you use Flaticon’s “attribution required” license, add the required credit text to your app’s About/Info section or README.
